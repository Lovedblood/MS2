# Running the project

## Runnig the backend

1. Change the directory to the backend folder

```bash
cd BACKEND
```

2. Install the dependencies

```bash
npm install
```

3. Run the server

```bash
npm start
```

## Running the frontend

1. Change the directory to the frontend folder

```bash
cd FRONTEND
```

2. Install the dependencies

```bash
npm install
```

3. Run the server

```bash
npm dev
```
